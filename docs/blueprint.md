# **App Name**: BizTrack

## Core Features:

- Revenue Tracking: Record and categorize all incoming revenue streams.
- Expense Tracking: Log and classify all outgoing expenses.
- Profit Calculation: Automatically calculate profit based on tracked revenue and expenses.
- Dashboard Visualization: Display key financial metrics (revenue, expenses, profit) on a customizable dashboard.
- Appointment Calendar: Schedule and manage appointments with date, time, and description. Display upcoming appointments on the dashboard.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to convey trust and stability.
- Background color: Light grey (#ECEFF1) for a clean, professional look.
- Accent color: Muted green (#81C784) to highlight positive financial metrics like revenue and profit.
- Body and headline font: 'Inter', a sans-serif font, for a modern, readable, and neutral appearance.
- Use simple, line-based icons for navigation and data visualization.
- Implement a clean, card-based layout for the dashboard and other key sections.
- Subtle transitions and animations when navigating between different sections of the app.
